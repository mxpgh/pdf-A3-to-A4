==========================================
  A3试卷转A4 转换工具 — 使用说明
==========================================

【功能】
将学校下发的A3扫描试卷（左右两页并排）自动拆分为单页A4格式，
方便家用打印机打印。

【使用步骤】
1. 双击运行 "A3转A4试卷转换" 程序
2. 拖入PDF文件或点击"选择PDF文件"
3. （可选）点击"预览分割线"检查自动检测效果
4. （可选）拖动滑块微调分割线位置
5. 点击"开始转换"
6. 转换完成后，在输出目录找到生成的 A4 PDF

【注意事项】
- 首次运行如果macOS提示安全问题：右键→打开→仍要打开
- 输入文件应为扫描版PDF（图片型）
- 转换后的PDF可直接用家用打印机打印
- 输出文件默认保存在原文件同目录，文件名添加"_A4"后缀

【常见问题】
Q: 分割线检测不准怎么办？
A: 点击"预览分割线"查看效果，拖动滑块手动调整后再转换。

Q: 支持Windows吗？
A: 支持。使用 Windows 版本的 PyInstaller 打包即可。

Q: 转换后图片模糊？
A: 默认300DPI渲染，已保证打印清晰度。
